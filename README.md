# 🤖 Puppeteer YouTube Automation 

A beginner-friendly automation project using **Puppeteer (Node.js)** that:

* Opens a browser
* Navigates to a YouTube channel
* Interacts with content (scroll, click video)
* Takes screenshots

---

## 🚀 Features

* 🌐 Open browser automatically
* 🔍 Search or directly open YouTube channel
* 📜 Auto scroll (human-like behavior)
* 🎥 Click and watch videos
* 👍 Attempt to like video
* 📸 Capture screenshots (channel + video)
* 🧠 Simulates real user actions

---

## 🛠️ Tech Stack

* Node.js
* Puppeteer

---

## 📦 Installation

1. Install Node.js: https://nodejs.org

2. Clone this repository:

```bash
git clone https://github.com/your-username/puppeteer-youtube-bot.git
cd puppeteer-youtube-bot
```

3. Install dependencies:

```bash
npm install puppeteer
```

---

## ▶️ Usage

Run the script:

```bash
node index.js
```

## ⚙️ How It Works

1. Launches Chrome browser
2. Opens YouTube channel (**FX MESSI**)
3. Scrolls the page
4. Clicks a video
5. Watches for a few seconds
6. Takes screenshots
7. Closes browser

---

## 📸 Output

* `fxmessi.png` → Full channel page
* `video.png` → Video page
